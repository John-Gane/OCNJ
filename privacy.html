Privacy information

Privacy notice

OCNJ.ETH is an independent website. This starter privacy notice describes the functions currently included in the static GitHub Pages site. Update it with legal counsel when payment, analytics, forms or other services are added.

Newsletter

The newsletter form is designed to submit an email address to Buttondown after the Buttondown username placeholder is replaced. Buttondown’s privacy practices and subscriber-management tools apply to information submitted through that form.

Location

The interactive map requests browser location only after a visitor selects “Use my location” or “Nearest parking.” The location is used in the visitor’s browser to center the map and calculate approximate distances. This starter code does not send that location to an OCNJ.ETH-owned server.

Maps and external services

The site loads map software and map tiles from third-party providers, including Leaflet, OpenStreetMap and optional Esri imagery. External links may open Google Maps, official business websites, government pages, Buttondown or payment providers. Those services may process technical and usage information under their own policies.

GitHub Pages

The website is hosted through GitHub Pages during the starter phase. GitHub may process visitor technical information for hosting and security purposes under its own privacy terms.

Payments

The OCNJ.ETH payment button can open a compatible cryptocurrency wallet. Any later card checkout should use a secure hosted payment provider such as Stripe rather than collecting card information directly on this static site.

Contact

Replace YOUR-EMAIL@gmail.com throughout the website before publishing so visitors have a working privacy contact.
