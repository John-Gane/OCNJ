# OCNJ CSS Repair

Replace the repository-root copies of:

- index.html
- style.css

with the files in this package.

The updated index.html loads:

- /OCNJ/style.css?v=20260722
- /OCNJ/data/map-locations.js?v=20260722
- /OCNJ/map.js?v=20260722
- /OCNJ/script.js?v=20260722

The version query forces Safari and GitHub Pages to request fresh files.
